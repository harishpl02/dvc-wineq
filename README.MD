create env

```bash
conda create wineq python=3.7 -y
```
to activate environment
```bash
conda activate wineq
```
To create requirements.txt file
```bash
ni requirements.txt
````
to install libraries from requirements.txt
```bash
pip install requirements.txt
```
To create READ.md file
```bash
ni requirements.md
````
